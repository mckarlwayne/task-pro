Task-Pro
A modern task management web application built with Laravel, designed to help users organize and track their tasks efficiently.

Features
Create and manage tasks with titles and descriptions
Mark tasks as complete or incomplete
View all tasks in a clean, responsive interface
User authentication for personalized task lists
Real-time updates and intuitive UI
